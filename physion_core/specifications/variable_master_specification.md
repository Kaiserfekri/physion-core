# Physion Master Variable Specification

Version: 1.0

Status: Stable

Purpose
-------

This document is the official registry of every variable used
inside the Physion Framework.

Every variable shall be defined exactly once.

Every implementation must follow this specification.

No variable may exist outside this registry.

---

# Variable Definition Rules

Every variable shall define:

- Variable ID
- Variable Name
- Description
- Owner
- State
- Category
- SI Unit
- Data Type
- Default Value
- Allowed Range
- Validation Rule
- Solver Owner
- User Level
- Industrial Source
- Scientific Source
- Status
